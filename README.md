# futhark-cli-average-floating-points

## Description
This is a POC project to demonstrate futhark. Requires typing into the terminal.

## Tech stack
- futhark
    - clang
    - lld
- nix

## Docker stack
- ubuntu:22.04

## To run
`sudo ./install.sh -u`
Example input: `[3.99,2.8,1.765]`

## To stop (optional)
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
